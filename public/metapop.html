Add C4EI to MetaMask
(MetaMask에 C4EI 추가)

Follow the steps below to add C4EI to MetaMask:
(C4EI를 MetaMask에 추가하려면 아래 단계를 따르세요.)

  1. Install and activate MetaMask on your browser if you haven't already.
    (아직 MetaMask를 설치하지 않았다면 브라우저에 설치하고 활성화하세요.)
  2. Click the MetaMask icon to open the popup window.
    (MetaMask 아이콘을 클릭하면 팝업창이 열립니다.)
  3. Select the "Network" tab in the popup window.
    (팝업 창에서 "네트워크" 탭을 선택합니다.)
  4. Click the "Add Network" button in the "Custom RPC" section.
    ("사용자 정의 RPC" 섹션에서 "네트워크 추가" 버튼을 클릭합니다.)
  5. Enter the following details:
    (다음 세부정보를 입력하세요:)
    • Network Name: C4EI
    • RPC URL: https://rpc.c4ei.net
    • Chain ID: 21004
    • Symbol: C4EI
    • Block Explorer URL: https://exp.c4ei.net

You can now use C4EI on MetaMask by adding C4EI tokens to your wallet and using C4EI DApps.
이제 C4EI 토큰을 지갑에 추가하고 C4EI DApp을 사용하여 MetaMask에서 C4EI를 사용할 수 있습니다.
